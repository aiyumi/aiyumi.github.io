Um Vídeo Introdutório ao CeVIO

Já mencionei aqui no blog alguns programas sintetizadores de voz, como o Espeak (que fornece a voz para o meu programa leitor de telas), o sintetizador de voz em Japonês Open JTalk (que o leitor de telas para Windows NVDA usa para ler em Japonês) e o software de síntese de canto VOCALOID, que inclusive comecei a usar há algum tempo. Existe um outro software sintetizador de voz Japonês, tanto para fala quanto para canto, chamado CeVIO. Hoje encontrei este vídeo (em Inglês) que fala sobre sintetizadores de voz japoneses e achei bem interessante. Ele começa introduzindo programas que vieram antes (MMDAgent e Open JTalk) e depois passa para o tópico principal, o CeVIO. Tem várias demonstrações das vozes (falando e cantando) e exemplos de como editar os parâmetros das vozes para ler textos em Japonês. (tem até dicas para quem está pensando em comprar o programa :D )

https://www.youtube.com/watch?v=k6c4p8Pjdf8
